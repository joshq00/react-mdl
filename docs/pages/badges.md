# Badges

## Demo

```jsx
{/* Number badge on icon */}
<Badge text="1">
    <Icon name="account_box" />
</Badge>

{/* Icon badge on icon */}
<Badge text="♥">
    <Icon name="account_box" />
</Badge>
```

```jsx
{/* Number badge on text */}
<Badge text="4">Inbox</Badge>

{/* Icon badge on text */}
<Badge text="♥">Mood</Badge>
```

## Configuration

| Prop     | Effect       | Remarks  |
|:---------|:-------------|:---------|
| text     | Assign string value to badge | Required |
